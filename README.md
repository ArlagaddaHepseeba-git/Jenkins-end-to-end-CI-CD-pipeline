# 🚀 End-to-End CI/CD Pipeline using Jenkins, Maven, SonarQube, Nexus, Tomcat & Slack

## 📌 Project Description
An automated end-to-end CI/CD pipeline built on AWS EC2 using Jenkins, 
integrating Git, SonarQube, Maven, Nexus, Tomcat and Slack to automate 
code integration, quality analysis, build, artifact management, 
deployment and team notifications.
